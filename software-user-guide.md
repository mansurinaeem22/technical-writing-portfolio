## Introduction
This document explains how users can use a Task Management Application
to create, update, and track tasks efficiently.

## Getting Started
After logging in, users are redirected to the dashboard where all tasks
are displayed.

## Creating a Task
1. Click **Create Task**
2. Enter task title and description
3. Select priority and due date
4. Click **Save**

## Editing a Task
1. Select an existing task
2. Click **Edit**
3. Update details
4. Click **Save**

## Deleting a Task
1. Select a task
2. Click **Delete**
3. Confirm deletion
